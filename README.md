# TinFlutter

##环境配置要求

基于Flutter2.2.3、Dart2.13.4版本开发，使用的三方库都适配了空安全。
目前可运行于Android iOS Web三种平台

### 需要安装的Plugin: 
- 基本配置插件: Dart、Kotlin、Flutter
- Flutter Intl 国际化插件，用于生成国际化相关代码
- Json to Dart 对象bean生成插件
- GetX GetX框架插件，用于生成GetX框架相关类和代码
